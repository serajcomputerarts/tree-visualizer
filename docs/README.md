# Tree Visualizer

A Python tool for converting parenthetical tree notation into beautiful visual representations.

## Features

- Parse parenthetical notation (e.g., `A(B,C)`)
- Generate high-quality tree visualizations (1920x1080)
- Support for complex nested trees
- Multiple visual themes
- Automatic layout optimization
- Export to PNG format
